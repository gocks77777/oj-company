# 숨고 견적 메시지 템플릿 & 견적 판단 기준 — O&J

> 숨고는 견적 보낼 때마다 캐시가 빠진다(단순견적 500~700원 / 상담연결 6,000~7,000원, **성사 안 돼도 소모**, 단 48시간 미열람 시 환불).
> → 핵심 원칙 2개: **① 맞는 요청에만 쏜다(정조준) ② 첫 두 줄에서 열람을 유도한다(후킹).**

---

## ⚖️ Part 1. "이 요청에 견적 쏠까?" 판단 기준

견적 보내기 전에 아래로 30초 점검. **GO 3개 이상이면 발송, STOP 1개라도 강하면 보류.**

### ✅ GO (쏴라)
- 요청 내용이 O&J 범위(업무 자동화 / 챗봇 / 문서 자동화 / 웹앱 / 로고송)에 명확히 맞음
- 예산이 적혀 있고 **현실적**(최소 20만원 이상 체감)
- 문제가 구체적 ("문의를 자꾸 놓쳐요", "이 작업이 반복돼요")
- 온라인 진행 가능 (지역 제약 없음)
- 우리 **실제 케이스와 거의 같은 패턴** → 거의 이긴 게임

### 🛑 STOP (아끼고 넘겨라 — 캐시 낭비)
- "단순 GPT 챗봇" 단가 경쟁만 붙는 요청 (차별점 0)
- 풀스택 + 디자인 + 기획 전부 떠넘기는 외주
- 단순 크롤링/데이터 수집
- 무기한 유지보수 전제, 예산 미기재 + "일단 견적부터"
- 예산이 비현실적으로 낮음 (5만원에 자동화 풀구축 등)

> 초반 캐시 예산: **2~3만원**으로 시작. 한 건 성사 = 캐시 본전 수십 배. 전환 안 되면 멘트/타깃을 바꿔라(캐시 더 붓기 전에).

## 🪝 Part 2. 견적 메시지 — 첫 줄이 생명

고객은 **첫 1~2줄만 보고 열람할지 결정**한다. 규칙:
1. 첫 줄 = 고객의 문제를 그대로 되짚기 (X "안녕하세요 저희는~")
2. 둘째 줄 = **숫자가 박힌 실제 결과** 한 방
3. 셋째 줄~ = 어떻게 할지 + 무료 진단 제안

---

### 템플릿 A — 업무/리드 자동화 요청

```
"문의·예약을 놓쳐서 새는 게 고민"이시라면 정확히 저희가 풀어온 문제예요.

실제로 광고 문의를 놓치던 1인 사무소에, 문의가 들어오면 즉시 담당자 폰으로
자동 알림이 가는 시스템을 '1일'에 구축했고 — 리드 확인 3~6시간 → 1분, 누락 0%,
월 운영비 0원으로 지금도 실사용 중입니다.

○○님 상황도 거의 같아 보여서, 어떤 도구를 쓰시는지(문의가 어디로 들어오는지)만
알려주시면 무료로 30분 진단해드릴게요. 견적은 진단 후 정확히 드리겠습니다.
```

### 템플릿 B — 챗봇 / AI 상담 요청

```
반복 문의에 일일이 답하느라 시간 뺏기는 부분, AI 챗봇으로 자동화할 수 있어요.

저희는 단순 GPT 붙이기가 아니라, ○○님 실제 데이터·FAQ를 학습시켜
"진짜 우리 업무를 아는 챗봇"으로 만듭니다. (실서비스 배포·운영 경험 있습니다.)

어떤 문의가 가장 자주 들어오는지 2~3개만 알려주시면, 무료 진단 때
바로 데모 방향을 잡아 보여드릴게요.
```

### 템플릿 C — 문서/보고 자동화 요청

```
같은 양식 문서를 매번 손으로 만드는 작업이라면, 입력 → 자동 생성으로 바꿀 수 있어요.

실제로 컨설팅펌에서 '입력값 → 사업계획서 + PPT 자동 생성'을 구축해
문서 작업 4~8시간을 10분대로 줄였습니다 (Claude AI + 자동 문서생성).

지금 그 문서를 어떤 정보로, 어떤 형식으로 만드시는지 샘플 하나만 보여주시면
자동화 가능 범위를 무료로 진단해드리겠습니다.
```

### 템플릿 D — 웹/앱 · 결제 서비스 구축 요청

```
아이디어를 '결제까지 되는 실서비스'로 빠르게 띄우고 싶으시면 맞는 팀이에요.

최근 결제 시스템 포함 온라인 강의 플랫폼을 기획부터 배포까지 '2주'에 단독으로
출시했고, 현재 실운영 + 유료 결제가 발생 중입니다.

○○님 서비스의 핵심 기능 1~2개랑 목표 일정만 알려주시면,
MVP 범위와 일정·견적을 무료 진단에서 잡아드릴게요.
```

### 템플릿 E — 로고송 / CM송 요청

```
브랜드에 딱 맞는 로고송, AI로 빠르게 여러 버전 만들어 골라드려요.

업종·분위기(밝게/차분하게 등)와 가게 이름만 주시면, 먼저 '데모'를 들려드리고
마음에 드는 방향으로 최종본을 납품합니다. 보컬/반주 분리본도 함께 드려요.

원하시는 느낌 한 줄만 적어주시면 샘플 방향 잡아 회신드리겠습니다.
```

### 템플릿 F — 모호한 요청 ("AI로 뭐 좀 하고 싶어요")

```
"AI로 업무를 줄이고 싶은데 뭐부터일지 모르겠다" — 이게 가장 흔하고, 저희가 제일 잘 푸는 시작점이에요.

무료 진단 30분에서 ○○님 하루 업무 중 '가장 시간 잡아먹는 반복 작업'을 같이 찾고,
자동화 가능한 것부터 우선순위를 잡아드립니다. (강요·비용 없습니다.)

지금 어떤 일을 하시는지, 어떤 게 제일 귀찮은지 한 줄만 적어주세요!
```

## 📌 Part 3. 견적 운영 팁
- **금액**: 숨고에선 처음부터 정가 박지 말고 "무료 진단 후 견적" + 범위 가격대만 언급. (낮게 박으면 단가경쟁, 높게 박으면 열람 안 함)
- **응답 속도**: 요청 뜨면 빠르게. 숨고 노출·선택률에 직접 영향.
- **열람 후 무응답**: 24시간 뒤 1회만 가볍게 팔로업("진단 일정만 잡아드릴까요?"). 그 이상 X.
- **성사 시**: 숨고페이 안전거래(3.5% 수수료)로 받으면 후기·분쟁 보호. 첫 거래는 안전거래 권장.
- **후기**: 끝나면 반드시 후기 요청. 첫 3건이 이후 모든 견적의 전환율을 좌우.
