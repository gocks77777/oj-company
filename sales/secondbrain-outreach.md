# 세컨드 브레인 세팅 — 영업 실행 (O&J)

> O&J 컴퍼니 (오정빈 + 정해찬) · AI 외주 2인 스튜디오
> 짝꿍 문서: 상품 정의는 `secondbrain-offer.md`, 전용 랜딩은 `docs/secondbrain.html`.
> 핵심 카피: **"까먹지 않는 사장님의 두 번째 뇌 + 디스코드 AI 직원팀"**
> 증거는 **우리가 직접 매일 쓴다는 것** 하나뿐 — 유료 케이스는 아직 없다. 그래서 영업 전략은 전부 **"우리가 직접 쓰는 걸 보여준다"**로 수렴한다.

## 1. 숨고 견적 메시지 — 첫 줄이 생명

> 상황: "세컨드 브레인 세팅", "AI로 업무·머릿속 정리", "노션/지식관리 자동화", "AI 비서 만들기" 같은 요청에 보낼 첫 견적.
> 규칙(숨고 공통): ① 첫 줄 = 고객 문제 되짚기 ② 둘째 줄 = 구체 결과/데모 제안 ③ 셋째 줄~ = 무료 진단. **정가 박지 말고 "무료 진단 후 견적".**

### 후킹 버전 A — "AI로 머릿속/업무 정리" 요청

```
"거래처 단가나 약속, 자꾸 까먹고 어디 적어놨는지 못 찾는 게 고민"이시라면
딱 저희가 푸는 문제예요.

저희가 만드는 건 한마디로 'AI가 읽는 사장님 메모장'이에요. 거래처 단가·약속·노하우를
넣어두면, 나중에 "그 집 단가 얼마였지?" 하고 물어보면 바로 답해줍니다. 저희 회사도
매일 이걸 쓰고 있어서, 무료 상담 때 실제 돌아가는 화면을 그대로 보여드릴 수 있어요.

지금 제일 자주 까먹거나 못 찾는 게 뭔지(단가? 약속? 아이디어?) 한 줄만 주시면,
30분 무료 상담에서 그것부터 넣어서 보여드릴게요.
```

### 후킹 버전 B — "AI 비서 / 챗봇으로 지식관리" 요청

```
그냥 챗봇이 아니라, ○○님 거래처·단가·노하우를 다 외워두고 물어보면 답해주는
'AI가 읽는 메모장'을 만들어 드려요.

거기에 카톡처럼 쓰는 AI 직원도 붙여드립니다. 폰으로 메시지 한 줄 보내면 정리·요약·초안을
대신 해줘요. (저희 회사가 지금 그렇게 일하고 있어요 — 상담 때 직접 보여드려요.)

평소에 "그거 어디 있더라" 하고 제일 자주 찾는 게 뭔지 1~2개만 알려주시면,
무료 상담에서 바로 보여드릴게요.
```

### 후킹 버전 C — "가게/사업 여러 개라 머리 복잡" 요청

```
이것저것 여러 개 벌여놓으시면, 일이 많다기보다 '머리 하나에 다 안 들어가는' 게
진짜 힘든 거잖아요. 그걸 AI한테 다 맡겨버리는 거예요.

다음에 켜면 "지난번에 여기까지 하셨어요" 하고 알려주니까 어디까지 했는지 안 까먹고,
뭐 떠오르면 말만 해두면 알아서 적어둡니다. 저희가 매일 그렇게 일하고 있어요.

지금 제일 자주 헷갈리는 게 뭔지 한 줄만 주시면, 30분 무료 상담에서
실제 화면으로 보여드릴게요.
```

> 운영 팁: 열람 후 무응답이면 24시간 뒤 1회만 가볍게 ("진단 일정만 잡아드릴까요?"). 캐시는 **세컨드 브레인/지식관리/AI비서 요청에만 정조준** — 단순 챗봇 단가경쟁 요청엔 쏘지 말 것.

## 2. 직접 영업 DM — 지인 사장님 / 여러 개 벌여놓은 사장님

> 톤: **영업 아님.** "신기한 거 만들었는데 보여드릴게요" / "한번 보실래요" — 파는 게 아니라 보여주는 것. 지인일수록 가격 얘기 먼저 꺼내지 말 것.

### DM 버전 1 — 지인 자영업 사장님

```
형/사장님, 저희가 요즘 'AI 메모장' 같은 걸 만들어서 회사에 쓰고 있는데요 —
거래처 단가나 약속을 AI가 다 외워둬서, 물어보면 바로 답해줘요. "그거 뭐였지" 하고
뒤질 일이 없어졌어요.

장사하시면서 머릿속에만 있는 거 자꾸 까먹거나 찾기 힘드시잖아요. 그거 딱 풀어주는
거라, 한번 보여드리고 싶어요. 파는 거 아니고 그냥 구경시켜드리는 거예요. 15분이면 돼요.
```

### DM 버전 2 — 가게/사업 여러 개 하는 지인

```
○○님 이것저것 여러 개 벌여놓으셨잖아요. 저도 그래서 머리 터질 것 같았는데, 요즘
하던 거를 AI한테 다 적어놨더니 "어제 어디까지 했지" 하고 헷갈릴 일이 없어졌어요.

카톡처럼 쓰는 AI 직원도 하나 붙여놨는데, 폰으로 시키면 정리·요약 알아서 해줘요.
저희가 매일 쓰는 거 그대로 한번 보여드릴게요. 마음에 드시면 ○○님 것도 만들어드리고요.
```

### DM 버전 3 — "머릿속 흩어진다" 입버릇인 지인

```
○○님 맨날 "정신없다, 까먹는다" 하시잖아요 ㅎㅎ 저희가 그거 풀어주는 걸 만들어서
직접 쓰고 있는데, 진짜 머리 정리되는 느낌이라 보여드리고 싶었어요.

뭐 떠오르면 말만 해두면 알아서 적어주고, 다음에 켜면 어디까지 했는지 딱 알려줘요.
15분만 보세요. 마음에 들면 ○○님 거 거의 공짜로 만들어드릴게요 (저희 첫 손님이라).
```

## 3. 케이스 없이 첫 고객 따는 법 — Results-First

> 후기 0건 상품의 진실: "남이 써봤다"는 증거가 없다. **그럼 우리가 한 명에게 공짜로/싸게 세팅해주고, 그 사람의 변화를 증거로 만든다.** 케이스를 기다리지 말고 **만든다.**

**1단계 — 베타 사장 1명 섭외 (1주차)**
- 지인 사장 중 위 DM에 가장 반응 좋은 1명을 고른다. "**무료 또는 베타가(예: 30~50만)로 세팅해드릴게요, 대신 2주 뒤 솔직한 후기 하나만 주세요**"로 제안.
- 조건: 흩어진 지식이 진짜 많은 사람(효과가 눈에 보일 사람). 효과 안 날 사람 고르면 케이스가 안 나온다.

**2단계 — 제대로 세팅하고 밀착해서 사용법 알려주기 (1~2주차)**
- 기존 카톡·엑셀·메모를 같이 한곳에 모으고, 자동 저장·자동 기억·디스코드 AI팀까지 풀세팅.
- "세팅하고 끝"이 아니라 **매일 쓰게** 만든다. 2주간 가볍게 챙긴다("오늘 뭐 물어보셨어요?").

**3단계 — 변화를 숫자·말로 뽑아낸다 (2주차 끝)**
- 후기 받을 때 막연히 "좋았어요" 말고 **구체 포인트**를 유도:
  "전에는 거래처 단가 찾느라 매번 카톡 뒤졌는데 이젠 물어보면 바로 나온다" /
  "**까먹는 게 확 줄었다**" / "아침에 어제 뭐 했는지 떠올리는 시간이 없어졌다".
- 가능하면 Before/After 한 줄씩 + 짧은 사용 영상/스샷(식별정보 가린 채).

**4단계 — 케이스화 → 다음 영업의 무기로 (3주차)**
- 이 후기를 `docs/portfolio.html` 케이스로, 숨고/크몽 첫 후기로, DM의 "한 분은 2주 만에 ○○ 효과 봤어요"로 재활용.
- **첫 케이스 1건이 생기면** 우리가 직접 쓴다는 증거 + 실고객 = 신뢰가 두 배. 그때부터 정가로 올린다.

> 핵심: 첫 1명은 매출이 아니라 **증거를 사는 비용**이다. 싸게/공짜로 줘도 케이스가 남으면 남는 장사.

## 4. 리드마그넷 — 최강 후킹 2종

세컨드 브레인은 말로 설명하면 안 와닿고, **보면 즉시 이해되는** 상품이다. 그래서 후킹은 두 개로 충분하다.

**① 무료 진단 30분**
- "사장님 업무에서 뭐가 제일 많이 새는지 같이 찾아드려요"로 문턱을 0으로.
- 질문 리스트는 `secondbrain-offer.md`의 7~10번 항목 사용. 진단 자체가 가치라 부담 없이 신청.

**② "우리가 직접 쓰는 시스템" 데모 — 진짜 무기**
- 진단 끝에 **O&J가 실제로 매일 쓰는 두뇌·디스코드 AI팀 화면을 라이브로** 보여준다.
- "이걸 사장님 것으로 그대로 복제해 드립니다"가 클로징. 후기 없어도 **눈앞에서 돌아가는 게 증거**.
- 데모 시나리오 3개를 미리 준비: ㉠ "거래처 단가 물어보면 바로 답" ㉡ "다시 켜면 어디까지 했는지 알려줌" ㉢ "카톡처럼 AI 직원에게 정리 시키기".

> 우리가 직접 매일 쓴다는 게 후기 부재를 이긴다 — "남이 좋대요"보다 "지금 제 화면에서 돌아갑니다"가 셀 때가 많다.

## 5. 전용 랜딩(`docs/secondbrain.html`) 활용

모든 영업의 **종착 링크**는 `docs/secondbrain.html` 하나로 통일한다. 숨고 견적·DM·진단 예약 직후 "자세한 건 여기서 보세요 → [링크]" 한 줄로 붙인다. 메시지는 짧게 후킹만 하고, 상품 구성·통증해결표·패키지·데모 설명 같은 무거운 정보는 전부 랜딩이 받게 해서 첫 메시지의 전환율을 지킨다. 랜딩에는 **"O&J가 직접 매일 쓰는 시스템"**임을 전면에 깔고(우리가 직접 쓴다는 게 곧 신뢰), 첫 케이스가 나오면 그 Before/After 후기를 랜딩 상단에 추가해 신규 리드가 진단을 신청하기 전에 이미 설득되도록 만든다. 링크 클릭은 곧 관심 신호 → 무료 진단 예약으로 바로 연결한다.
