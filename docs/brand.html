05 · Usage

사용 가이드

최소 여백과 금지 사항. 일관되게 쓸수록 신뢰가 쌓입니다.

최소 여백 = 심볼 높이의 1/4 (= "O" 굵기)

DO — 권장

  • 다크 배경엔 컬러/화이트 로고
  • 밝은 배경엔 모노(검정) 또는 잉크 패널 위 컬러
  • 최소 크기: 심볼 24px, 풀 로고 너비 120px 이상
  • 충분한 여백 확보 (위 가이드)

DON'T — 금지

  • 그라데이션 색·각도 변경
  • 로고 비율 왜곡 · 회전
  • 저대비 배경(보라/파랑 위 컬러 로고)
  • 심볼과 워드마크 임의 재배치
  • 그림자·외곽선·효과 추가

06 · Export

PNG로 export 하는 법

SVG는 벡터라 무한 확대됩니다. 래스터(PNG)가 필요할 때 — 가장 쉬운 순서대로.

방법 A · 브라우저 (도구 설치 0)

SVG 파일을 브라우저에 드래그해서 열고 → 확대 후 캡처. macOS는 ⌘⇧4(영역) / ⌘⇧5(녹화·고정). 빠르지만 화면 해상도에 묶입니다.

방법 B · macOS 기본 내장 (qlmanage)

추가 설치 없이 고해상도 PNG 생성:

# 512px 폭으로 현재 폴더에 PNG 생성
qlmanage -t -s 512 -o . logo-symbol.svg
# 결과: ./logo-symbol.svg.png
방법 C · rsvg / Inkscape (정밀, 권장)

정확한 픽셀·투명 배경이 필요할 때. 한 번만 설치:

# 설치 (Homebrew)
brew install librsvg          # rsvg-convert
brew install --cask inkscape  # 또는 Inkscape

# rsvg-convert — 폭 1024px, 투명 PNG
rsvg-convert -w 1024 logo-symbol.svg -o logo-symbol@1024.png

# Inkscape — DPI 지정 export
inkscape logo-full.svg --export-type=png --export-dpi=300 \
  --export-filename=logo-full@300.png
방법 D · 온라인 변환

설치가 어려우면 CloudConvert / svgtopng.com 에 업로드. 단, 로고는 외부 업로드를 피하는 게 안전하니 가능하면 A~C를 쓰세요.
