## 클라이언트 개요

| 항목 | 내용 |
|---|---|
| 업종 | 리드 기반 SMB (1인 사무소) |
| 규모 | 1인 운영 |

## Before (진단)

**상황:**
- Meta(페이스북/인스타) 광고를 직접 운영 중
- 광고 문의가 하루 수십 건씩 들어오지만, 메신저 알림을 놓치거나 늦게 확인
- 문의 → 상담 전환까지 평균 3~6시간 지연
- 현장 이동 중일 때 리드를 실시간으로 확인할 방법 없음

**비용:**
- 광고비 지출 대비 응답 지연으로 리드 손실 발생
- 1인 운영이라 별도 직원 고용 불가

---

## Solution (설계 및 구현)

**아키텍처:**
```
Meta Ads (잠재고객 양식)
    ↓
Make (구 Integromat) — Webhook 트리거
    ↓
데이터 파싱 + 포맷팅 (이름, 연락처, 관심 항목)
    ↓
Telegram Bot → 클라이언트 전용 채널에 실시간 알림
```

**핵심 구현 포인트:**
- Meta Ads 잠재고객 양식(Lead Form)과 Make Webhook 연동
- 리드 데이터를 파싱해 "이름 / 연락처 / 문의 내용" 포맷으로 자동 정리 (광고 출처는 Meta hidden field 설정 시 추가 가능)
- Telegram Bot을 통해 클라이언트 스마트폰에 즉시 푸시
- 현장에서도 1분 내 확인 + 바로 전화 가능

**사용 도구:**
- Meta Ads (잠재고객 양식)
- Make (자동화 파이프라인)
- Telegram Bot API
- 추가 필터링 로직 (중복 리드 제거)

**구현 기간:** 1일

## AX 인사이트

**왜 이게 단순 자동화가 아니라 AX인가:**
1. **비즈니스 프로세스 재설계** — "광고 → 수동 확인" 플로우를 "광고 → 자동 알림 → 즉시 대응"으로 전환
2. **ROI 즉시 측정 가능** — 리드 손실 0% + 운영비 0원
3. **1인 사업자의 경쟁력 = 대기업 수준 응답속도** — AI 자동화로 인력 없이 구현
4. **지속 운영 중** — 일회성 프로젝트가 아닌 실 비즈니스 인프라

**재현 가능성 (컨설팅 패키지로 확장 시):**
- 동일 패턴을 학원, 인테리어, 피부과 등 "리드 기반 SMB" 전체에 적용 가능
- 구축 1일, 월 유지비 0~$10, 즉각 ROI — 패키지화 가능
