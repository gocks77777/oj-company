# 가격 및 수임범위 (내부 운영 문서)

> 첫 인바운드 문의 시 즉답할 수 있도록 사전 정의. 공개 페이지에는 노출하지 않음.
> 가격은 "진단 무료 → 견적" 게이트로 운영.

---

## 제공 서비스 3종 단가 기준

### ① 업무 자동화 직접 구축

| 규모 | 내용 | 단가 범위 |
|---|---|---|
| 소형 (1~2일) | 리드 파이프라인, 알림 자동화 (Make + Telegram 등) | 30~50만원 |
| 중형 (3~5일) | 문서 자동화, 다단계 파이프라인 (Claude API + FastAPI) | 80~150만원 |
| 대형 (1~2주) | 웹앱 포함 풀 자동화 시스템 | 200~400만원 |

> 초기 레퍼런스 확보 목적이면 소형 1건은 50% 할인 또는 무료로 진행 가능 (단, 케이스 스터디 공개 동의 필수).

### ② AX 전략 컨설팅

| 형태 | 내용 | 단가 |
|---|---|---|
| 무료 진단 30분 | 현재 업무 병목 파악 + 자동화 가능 항목 체크 | 무료 (첫 상담) |
| 단발 컨설팅 | 자동화 우선순위 진단 + 실행 로드맵 문서 | 20~30만원 |
| 월 리테이너 | 지속 자문 + 구현 지원 (월 4회 미팅) | 50~100만원 |

### ③ AI 강의 / 워크샵

| 형태 | 내용 | 단가 |
|---|---|---|
| 단발 강의 (1~2시간) | Claude Code / Make / AI 자동화 입문 | 30~50만원 |
| 반일 워크샵 (4시간) | 실습 포함, 기업/학교 대상 | 80~120만원 |
| 동아리/학생 | 비영리 목적 | 무료~10만원 (교통비) |

> 출장 강의(오프라인)는 수도권 기준. 지방 출장비 별도.
> 온라인(Zoom/Google Meet)은 지역 무관, 단가 동일.

## 2인 체제 활용 (O&J)

> O&J는 오정빈·정해찬 2인 스튜디오. 단가는 위 기준을 따르되, 2인 분업으로 처리량/품질을 높인다.

- **병렬 처리:** 구축 작업과 문서/케이스 정리를 분담해 동일 기간 내 처리량 확대
- **2-트랙 진행:** 한 명이 구축, 한 명이 클라이언트 커뮤니케이션/진단을 맡아 리드타임 단축
- **강의/워크샵:** 2인 진행 시 강사-실습보조 구성으로 인원 많은 워크샵 대응 가능
- **대형 건:** 웹앱 풀 자동화처럼 범위가 큰 건은 2인 동시 투입으로 일정 단축

## 거절 템플릿 (시간 도둑 방지)

**거절해야 하는 의뢰:**
- 단순 GPT 챗봇 구축 (차별점 없음, 단가 경쟁만 발생)
- 풀스택 외주 개발 (디자인 포함, 기획 포함)
- 크롤링/데이터 수집 단순 작업
- 무기한 유지보수 계약 (별도 협의 없는 경우)

**거절 문구 예시:**
> "말씀해주신 작업은 저희 O&J가 집중하는 AI 자동화/AX 컨설팅 범위를 벗어납니다. 더 잘 맞는 분을 소개해드릴 수 있을 것 같습니다."

## 첫 문의 응대 흐름

```
1. 문의 접수 (이메일/카카오)
    ↓
2. 48시간 내 답변: "무료 진단 30분 일정 잡겠습니다"
    ↓
3. 무료 진단 30분 (Zoom 또는 카카오 통화)
   - 현재 어떤 업무가 병목인지
   - 예산/일정 대략 파악
    ↓
4. 견적서 발송 (48시간 내)
    ↓
5. 계약 or 거절
```

## 세무/사업자 처리

- 강의/컨설팅 수입 발생 시: **프리랜서 3.3% 원천징수** (사업자등록 전)
- 월 수입이 일정 규모 초과 시 간이과세자 사업자등록 검토
- 인보이스 양식: 상호/서비스 내용/금액/계좌번호 포함 PDF (카카오페이/계좌이체)

## 초기 레퍼런스 전략

8주 내 1건 성사가 목표. 우선순위:
1. **지인/네트워크** — 아는 스타트업/창업팀
2. **인바운드** — 콘텐츠 발행 후 문의 온 첫 번째 의뢰인
3. **할인 제공** — 첫 건은 소형 패키지 50% 할인 + 케이스 스터디 공개 동의 조건
