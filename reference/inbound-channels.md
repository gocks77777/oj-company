# 인바운드 채널 설정 (한국 컨설팅/강의 타겟)

> 목표: 한국 중소기업·기업 교육 담당자가 유입되어 컨설팅/강의 문의를 보내는 경로 구축

---

## 채널 구조

```
콘텐츠 게시물 (노출)
    ↓
포트폴리오/소개 사이트 AX 섹션 (신뢰 구축)
    ↓
연락처 클릭 → 이메일 / 오픈카톡
    ↓
상담 요청 접수
```

---

## 1. 소개 사이트 AX 섹션

### 섹션: "AX 컨설팅 / 강의 의뢰"

**섹션 내용:**
```
[헤딩] AI로 업무를 바꾸고 싶으신가요?

[서브] 반복 업무 자동화부터 팀 AI 교육까지
실제 ROI를 만드는 방식으로 접근합니다.

[제공 서비스 카드 3개]
┌─────────────────────┐  ┌─────────────────────┐  ┌─────────────────────┐
│ 🔧 업무 자동화 구축   │  │ 📊 AX 전략 컨설팅    │  │ 🎓 AI 도구 강의/워크샵│
│                     │  │                     │  │                     │
│ 리드 자동화, 문서     │  │ 어떤 업무를 먼저      │  │ Claude Code, Make,  │
│ 생성, 알림 파이프라인  │  │ 자동화해야 하는지     │  │ AI 자동화 실습 교육  │
│ 등 직접 구축         │  │ 진단 + 로드맵         │  │ (기업/학교/동아리)   │
└─────────────────────┘  └─────────────────────┘  └─────────────────────┘

[사례 미리보기]
• 리드 기반 SMB 자동화: 응답 시간 3~6시간 → 1분 (실사용 중)
• AX 컨설팅 문서 자동화: 사업계획서 작성 4~8시간 → 10분 (목표, MVP 개발 중)

[CTA 버튼]
[📩 의뢰/문의하기] → mailto (AX 컨설팅 문의 제목)
[💬 카카오톡 오픈채팅] → (오픈채팅 링크)
```

### 구현 방법
- 소개 사이트에 새 섹션 컴포넌트 추가
- 케이스 스터디 공개 페이지 링크 (case-lead-automation.md 내용을 노션 등에 게시)
- 재배포

---

## 2. 연락처 채널 정비

### 이메일
- O&J 대표 문의 이메일 — 유지
- 자동응답 설정: "문의 감사합니다. 영업일 기준 48시간 내 답변드리겠습니다."

### 카카오톡 오픈채팅
- 오픈채팅방 생성: "O&J AI 자동화 상담"
- 링크 생성 후 소개 사이트 + 프로필에 추가
- 장점: 국내 클라이언트가 이메일보다 편함

### 프로필 DM
- 소개 마지막에 "강의/컨설팅 의뢰 → DM 또는 문의 이메일"
- 'Contact info'에 이메일 노출 설정 확인

---

## 3. 케이스 스터디 공개 페이지

**목적:** Featured/소개에 붙일 수 있는 공개 URL

**만들 페이지:**
1. `O&J AI 자동화 케이스 스터디` (메인 허브)
   - 케이스 #1: 리드 기반 SMB 리드 자동화 (case-lead-automation.md 내용)
   - 케이스 #2: AX 컨설팅 문서 자동화 (case-doc-automation.md 내용, MVP 완성 후)
2. 각 케이스: Before/Solution/After 포맷

**공개 설정:** 검색엔진 색인 허용 ON

## 실행 순서 (우선순위)

| 순서 | 액션 | 소요 시간 | 임팩트 |
|---|---|---|---|
| 0 | **클라이언트 공개 동의 확보** (카카오 1줄: "업종+인용 공개 동의합니다") | 즉시 | 케이스 발행 선행 조건 |
| 1 | 소개 프로필 Headline+About 정비 | 30분 | 즉시 노출 |
| 2 | 케이스 스터디 게시물 발행 (동의 확보 후) | 1시간 | 즉시 인바운드 |
| 3 | 카카오 오픈채팅 생성 + 프로필에 링크 추가 | 30분 | 국내 클라이언트 |
| 4 | 노션 케이스 스터디 페이지 게시 | 1시간 | Featured용 |
| 5 | 소개 사이트 AX 섹션 추가 + 재배포 | 2~3시간 | 장기 신뢰 구축 |
| 6 | AX 앱 데모 GIF → 콘텐츠 게시물 | MVP 완성 후 | 가장 강한 증거 |

## 성공 지표 (8주 후)

| 지표 | 측정 방법 | 목표 |
|---|---|---|
| 게시물 노출수 | 게시물 인사이트 | 1편 기준 1,000+ |
| 게시물 반응 | 좋아요 + 댓글 수 | 1편 기준 20+ |
| DM/이메일 문의 | 직접 수신 카운트 | 월 2건 이상 |
| 강의/컨설팅 성사 | 계약 or 레퍼런스 확보 | 8주 내 1건 |
| 주당 처리 가능 의뢰 | 자체 점검 | 2인 기준 최대 3~4건 (초과 시 일정 조율) |

**미달성 시 stop-loss:** 12주에도 문의 0건이면 게시물 타겟팅·해시태그·후크 전면 재검토
